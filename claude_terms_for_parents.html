ターミナル（黒い画面）で動くエージェント。

その分、最初のセットアップに少し勉強が必要です💻

こんな指示ができる 「子どもの写真フォルダを月ごとに整理するスクリプトを作って、毎晩自動で動くようにして」

非エンジニアならCoworkで十分すごいです🌷

Chapter 3

MCPとAPI、
結局なにが違うの？

いちばんモヤモヤするやつ、解決します
🔌 ひとことで言うと

APIは「個別の専用ケーブル」🔌
MCPは「USB-Cの統一規格」📱

MCPはAPIを置き換えるんじゃなく、
APIの上に立つ共通ルールです

📞 API

アプリ同士が話す「電話線」

APIは Application Programming Interface の略。
アプリAとアプリBがお話しするための専用の電話線みたいなもの。

例：天気アプリは「気象庁API」に電話して天気を聞き出してます☎️

困っていたこと アプリごとに電話のかけ方が違うので、つなぐ相手が増えるたびに専用ケーブルを作る必要があった。これがエンジニアの大変なところでした💦
🔌 MCP

AIのための「USB-C」

MCPは Model Context Protocol の略。
Anthropic社が2024年11月に発表した共通の差込口の規格です✨

これにより、AIは「同じ差し方」で色んなツールにつながれるようになりました🌸

たとえばこんな感じ 以前：Notion用、Google Drive用、Slack用…とつなぎ方を全部別に勉強が必要

MCP後：全部同じやり方でつながる🎀
📱 ClaudeなどのAI

🔌

📊 ザックリ比較表

使う人

あれがMCPの実装そのものです✨
難しいコードを書かずに、ボタンひとつでつなげるのはMCPのおかげなんです🎀

Chapter 4

Git と GitHub
ってなに？

エンジニアじゃなくても、これは知っとくとお得
📔 ひとことで言うと

Gitは「成長記録ノート」📔
GitHubは「ノートを保管する共有書庫」🏛️

子どもの成長記録を、家族で
共有できるイメージです🌸

📔 Git

ファイルのタイムマシン

Gitは、ファイルの変更履歴を全部記録してくれるツール。
「あ、3日前のこのバージョンに戻したい！」を可能にします⏰

育児で例えるなら 離乳食の献立記録：「3月10日のメニューに戻したい」「先週のチャレンジレシピをもう一度作りたい」が完璧にできる📔
🏛️ GitHub

ノートを置く共有書庫

GitHubは、Gitで記録したノートをクラウドに保管するサービスです。
自分のPCが壊れてもデータが消えない＋他の人と共有できるのが特徴🌸

こんなメリット ・PCが壊れてもデータが安心
・スマホ・タブレット・PCどこからでも見られる
・夫婦で同じファイルに書き込める
・世界中の人が作った無料の「育児テンプレ」が見られる
💡 なぜ知っておくと得？

Claude Codeと相性バツグン

Claude CodeはGitと組み合わせて使うのが基本スタイル。
AIがコードを書く→Gitで記録→失敗したらすぐ戻せる、という安心感があります💕

また、ネット上には「うちの献立アプリ」「うちの家計簿」のような無料のテンプレがGitHubにたくさん公開されていて、それを自分用にカスタマイズして使えます✨
